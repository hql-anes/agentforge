# AgentForge Workflow

AgentForge Workflow 是一个面向 AI Native 应用开发者、中小型技术团队与自动化运营场景的多 Agent 智能研发工作流平台 Demo。

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`。

## 构建检查

```bash
npm run build
```

## Vercel 部署

- Framework Preset: Next.js
- Root Directory: ./
- Install Command: npm install
- Build Command: npm run build
- Output Directory: 留空
